# x64dbg-Tutorial-to-understand-packers
an interactive x64dbg simulator right here that mimics the real thing.Each step shows you exactly what changes in registers and memory — mimicking what you will see in real x64dbg. Once this flow feels natural, doing it in the real debugger will make complete sense.

This will guide me to Load a UPX-packed binary in x64dbg. Set a breakpoint on VirtualAlloc. Step through until the OEP. Dump the unpacked binary using Scylla.
